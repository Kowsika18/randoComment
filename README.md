# RandoComment

App to update user's Comment in the Portal

## Usage
1. Backend - Express, Node JS
2. Frontend - HTML, CSS, JS
3. Database - JSON, LocalStorage

## How to Run the App ?

Open it in Terminal or any Code Editor.
Type in the comment

```bash
npm run dev
```
